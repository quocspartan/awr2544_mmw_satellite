## Features
- 76GHz to 81GHz mmWave sensor
- 4TX/4RX 3D waveguide antenna
- On-chip Arm R5F controller
- On-chip Hardware Accelerator 1.5 (HWA 1.5)
- Direct interface with DCA1000EVM
